System Upgrade

            

            

                
A. Online upgrade (host has internet)

                

                
sudo bash /opt/jt-glogarch/deploy/upgrade.sh


                
B. Offline / air-gapped upgrade

                

                
# 1) On an internet-connected machine (same Python + CPU arch as target):
bash scripts/build-offline-bundle.sh

# 2) Carry dist/jt-glogarch-<ver>-offline.tar.gz to the target host (USB/scp)

# 3) On the target host (root):
tar xzf jt-glogarch-<ver>-offline.tar.gz
cd jt-glogarch-<ver>-offline
sudo bash upgrade-offline.sh
